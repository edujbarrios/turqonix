# Turqonix

A dark, turquoise-centric VS Code theme built around the brand palette of **edujbarrios.com**.

## Color Palette

| Role | Hex |
|---|---|
| Background | `#0f172a` |
| Foreground | `#e5e7eb` |
| Keyword | `#40E0D0` |
| Function | `#66E6DD` |
| String | `#8CF0EB` |
| Type / Class | `#B2FAF9` |
| Comment | `#6b7280` |
| Status Bar | `#004d4d` |

## Design Goals

- WCAG-friendly contrast for long coding sessions
- Single coherent accent: turquoise
- Minimal visual noise
